### 🚀 Projeto em Destaque: Sentinel EDR

Desenvolvedor Chefe do **Sentinel EDR**, um Endpoint Detection and Response construído do zero em Python para ambientes Linux. Um projeto que consolida todos os meus estudos de Defesa Ativa.

**Funcionalidades de Elite Desenvolvidas:**
- **File Integrity Monitoring (FIM)** com cálculo heurístico de Entropia de Shannon (Prevenção contra Ransomware/Criptografia).
- **YARA Memory Scanning:** Escaneamento da memória RAM (Heap/Stack) dos processos suspeitos em tempo real para abater **Fileless Malware**.
- **Suricata Network IPS:** Integração com o NetfilterQueue no Linux para dropar pacotes maliciosos no nível do Kernel de forma automatizada.
- **Engenharia Reversa no `/proc`:** Mapeamento inteligente de Process Trees (Processos-Pai e Filhos) rastreando Inodes para caçar backdoors (Reverse Shells).
- **C2 & Telemetria:** Painel de controle assíncrono via API Flask e Bot do Telegram para Exterminação Remota de Ameaças (Isolamento/Kill/Shredding).
